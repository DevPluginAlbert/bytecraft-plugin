# ByteCraft Plugin Studio — Roadmap

**Company Goal:** Build Revenue-Generating Minecraft Plugin Portfolio
**Target:** £100/month revenue on BuiltByBit marketplace
**Strategy:** Ship small, focused plugins (one at a time). MVP in ≤7 days. Zero feature bloat.

## Phase 1: Foundation (Days 1–3)

**Objective:** Establish the studio's development infrastructure and pick Plugin #1.

- [ ] Initialize studio GitHub repo with project structure
- [ ] Set up CI/CD and branch protection
- [ ] Complete team onboarding review and hire any missing roles
- [ ] Audit Farmable and InvSafe for bug fixes, documentation gaps, and listing improvements
- [ ] Research and select Plugin #1 target (market gap analysis on BuiltByBit)

**Deliverable:** Studio fully operational, Plugin #1 scoped

## Phase 2: Plugin #1 MVP (Days 4–10)

**Objective:** Ship MVP of first new revenue plugin.

- [ ] Create plugin #1 design doc and architecture
- [ ] Implement MVP (≤7 days)
- [ ] Code review and QA pass
- [ ] Write documentation (README, setup guide, wiki)
- [ ] Create BuiltByBit listing assets
- [ ] Launch and monitor feedback

**Deliverable:** Plugin #1 live on BuiltByBit

## Phase 3: Validation & Plugin #2 (Days 11–21)

**Objective:** Validate Plugin #1 reception, begin Plugin #2.

- [ ] Review Plugin #1 sales data, reviews, and customer questions
- [ ] Ship critical bug fixes based on early feedback
- [ ] Research and scope Plugin #2
- [ ] Implement Plugin #2 MVP
- [ ] Code review and launch

**Deliverable:** Two new plugins on marketplace (4 total including legacy)

## Phase 4: Scale to £100/mo (Days 22–45)

**Objective:** Third plugin + optimization.

- [ ] Analyze revenue trajectory
- [ ] Identify Plugin #3 from highest-demand marketplace category
- [ ] Implement Plugin #3 MVP
- [ ] Optimize BuiltByBit listings (conversion rate improvements)
- [ ] Iterate on all active plugins based on customer feedback

**Deliverable:** £100/mo target on track

## Ongoing: Quality & Maintenance

- Code review on every plugin before release
- Documentation always ships with the plugin
- Weekly backlog grooming
- Customer feedback loop from BuiltByBit reviews → Paperclip issues

## Plugin Pipeline (candidate backlog)

These are candidate categories for future plugins. Priority finalized via market research.

1. **Admin Tools** — moderation helpers, auto-punishment, chat control
2. **Automation** — auto-backup, scheduled tasks, world management
3. **Monetization** — donation perks, custom shop GUIs, player economy tools
4. **Quality of Life** — player conveniences servers pay for (warps, homes, teleport)
5. **Anti-Grief** — lightweight protection (complement to existing plugins)
